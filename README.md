# 🦅 JosaeWorld

> SOOP 스트리머 **시조새 크루** 팬 앱

---

## 소개

SOOP 플랫폼의 시조새 크루 방송 상태를 실시간으로 확인할 수 있는 팬 앱입니다.

## 기술 스택

| 분류   | 라이브러리                                 |
|------|---------------------------------------|
| UI   | Jetpack Compose, Lottie, Glide        |
| 아키텍처 | MVVM, Clean Architecture, Hilt        |
| 네트워크 | Retrofit, OkHttp                      |
| 비동기  | Coroutines, Flow                      |
| 백엔드  | Firebase Realtime DB, Cloud Functions |
